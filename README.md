Parents or teachers can give a list of addition/subtraction problems, and the function prints them neatly in a column format – with or without answers. This is great for worksheets, homework, or flashcards.
When show_answers=True, you get the correct answer aligned below each problem. You can use it to verify your own calculations or to create self-checking exercises.
This code is a great example of string alignment (rjust), error handling (too many problems, invalid operators, non-digit characters, number limits), and conditional output. You can use it to teach or learn these concepts.
It’s the exact problem from freeCodeCamp’s “Arithmetic Formatter”. You can test your solution against their requirements, or use it as a reference for similar coding challenges.
